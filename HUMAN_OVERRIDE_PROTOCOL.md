# Human Override Protocol

## When human_required is MANDATORY
- Production promotion
- Hardcoded driver replacement
- Guard removal
- Evidence cleanup
- New GPT conversation
- Base URL fallback
- Scope expansion beyond authorization

## Agents MUST NOT Decide Alone
- GPT_ACCEPTED / FINAL_ACCEPTED
- REAL_CHAIN_UNBLOCKED
- PRODUCTION_APPROVED
- GUARD_REMOVAL_APPROVED

## Who Must Approve
- Current authorized GPT for review decisions
- Human user for production/boundary changes

## When Automation MUST Stop
- GPT rejected/blocked/human_required
- REVIEW_RUN_ID mismatch
- review_unverified
- CDP unavailable
- URL mismatch
